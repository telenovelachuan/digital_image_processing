Demo Code for the Paper: "Minimum Barrier Salient Object Detection at 80 FPS"


System Requriement: 64-bit Windows 7 or above


Usage:

1. Unzip all the files, including all the dlls and the subfolders "output" and "src".
2. Run runMB.bat for MB, and runMB+ for MB+ (see the descriptions of these two methods in our paper).
3. The output saliency maps are saved in the folder "output".
4. If you would like to test your own images, please put them in the folder "src", and run the bat files again.


The code has been tested on a computer with Intel Xeon CPU W3670 @3.2GHzX2, 12GB RAM.
